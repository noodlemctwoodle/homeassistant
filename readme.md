### Home Assistant Hardware Configuration

Sever infrastructure 

- AMD Ryzen 3700X
- 64GB Corsair 3200mhz
- 50TB Rotational Drives
- 8TB SSD Cache
- NVIDIA Quadro P2000
- Quad Port Intel NIC
- Broadcom 9800i 16 port SAS Controler
